RO.ME "3 DREAMS OF BLACK"

RO.ME is an innovative and immersive interactive film with great music. It uses creative and colorful models in the film. It also embraces collabration as other people can create models and add it to the dream for others to enjoy.

Technology: The main component used is WebGL which is a JavaScript API for hardware accelerated rendering on a browser. The experience was made possible by using simple but powerful tools like HTML5, JavaScript and WebGL.
